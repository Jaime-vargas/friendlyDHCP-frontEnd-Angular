About

Friendly DHCP

It is a web application developed in Angular (front end) and Java (back end), designed to simplify the creation and management of DHCP configuration files. It allows users to navigate between different sections (Networks, Devices, Settings, About) through an interactive sidebar, and dynamically view the content of each section in a main area. The app aims to streamline tasks that are usually done manually in text files, providing a visual interface to define networks, assign devices, and generate configurations quickly and reliably.
